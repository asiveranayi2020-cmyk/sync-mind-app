# AI Workplace Productivity Assistant

A modern, responsive SaaS-style web application that helps professionals automate workplace tasks using Artificial Intelligence. Built to demonstrate practical AI implementation, strong prompt engineering, workplace productivity enhancement, responsible AI usage, and modern UI/UX design.

🔗 **Live Demo:** https://sync-mind-app.lovable.app

---

## ✨ Features

### 🧠 AI-Powered Productivity Tools

- **Smart Email Generator** — Draft professional workplace emails tailored to your audience and tone (formal, friendly, persuasive, concise).
- **Meeting Notes Summarizer** — Turn long, messy meeting notes into structured summaries with key decisions and clear action items.
- **AI Task Planner** — Prioritize tasks and generate a realistic, time-boxed daily or weekly schedule.

### 🎨 Modern SaaS Dashboard

- Clean, professional interface inspired by leading SaaS productivity platforms
- Collapsible sidebar navigation with iconography
- Fully responsive layout (mobile, tablet, desktop)
- Light & dark mode with system preference detection
- Polished components powered by shadcn/ui and Tailwind CSS v4

### 🛡️ Responsible AI

- Visible Responsible AI disclaimer across every tool
- User-verification reminders before acting on AI output
- Transparency about model limitations and potential inaccuracies
- All AI output is editable — the user remains in control

## 🧩 Prompt Engineering

Each tool ships a carefully crafted **system prompt** that:

- Establishes a clear professional role for the AI
- Constrains output to a predictable, structured format (Markdown sections, action items, time blocks)
- Honors user context: audience, tone, priorities, deadlines
- Avoids hallucinated facts by working only from the user-provided input

This delivers consistent, high-quality, context-aware responses suitable for real workplace use.

## 🏃 Getting Started

### Prerequisites

- [Bun](https://bun.sh/) (recommended) or Node.js 20+

### Installation

```bash
# Clone the repo
git clone <your-repo-url>
cd <project-folder>

# Install dependencies
bun install

# Start the dev server
bun run dev
```

The app will be available at `http://localhost:8080`.

### Environment Variables

The AI features call the Lovable AI Gateway and require:

```
LOVABLE_API_KEY=your_api_key_here
```

When running on Lovable Cloud, this is provisioned automatically.

## 🛡️ Responsible AI Statement

This application uses Artificial Intelligence to assist users with workplace productivity tasks. AI-generated content may contain inaccuracies and should be **reviewed and verified** before use. Users remain responsible for final decisions, communications, schedules, and actions generated by the system.

We do not persist user inputs — prompts are sent to the AI model only to generate the requested response.
